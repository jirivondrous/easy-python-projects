# 🐍 Easy Python Projects

This repository contains beginner-level Python projects I'm working on as I learn Python.  
Each script focuses on different programming concepts like conditionals, loops, functions, and input/output.

## 📦 Projects

- `rock_paper.py`: A simple rock-paper-scissors game
- `hangman.py`: Classic hangman game with ASCII art
- `guess.py`: Number guessing game with randomization
- More to come as I continue learning and improving!

- Improve code readability, structure, and logic over time
- Learn Git and GitHub workflows by committing regularly

## 🔧 Tech Stack

- [Visual Studio Code](https://code.visualstudio.com/) as editor
- Git + GitHub for version control
